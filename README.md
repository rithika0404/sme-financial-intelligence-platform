# SME Financial Intelligence Platform

An AI-powered financial health assessment tool for SMEs built for hackathon submission.

## Features
- Financial KPI Dashboard (Revenue, Profit, Debt Ratios)
- Creditworthiness Scoring + Rating
- AI Revenue Forecasting (Next 3 Months)
- Compliance & GST Metadata Check
- Working Capital Optimization Insights
- Bank/NBFC Product Recommendations
- Automated Bookkeeping Assistant
- Investor-ready PDF Report Generation

## Input File Requirements
Upload CSV/XLSX with at least:
- Month, Revenue, Expenses, Debt

Optional:
- Receivables, Payables, Inventory, Tax

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Demo Data
Use:
- sample_financials_12months.csv

This platform provides SMEs with instant credit risk assessment, forecasting, compliance readiness, and actionable financial product advisory.
